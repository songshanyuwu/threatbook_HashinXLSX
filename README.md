# threatbook_HashinXLSX

该项目源自https://github.com/zyzyzzyyy/threatbook_MD5inXLSX    向原作者表示感谢。

该项目与原代码相比，更新API接口的地址，以及对返回结果的处理。

具体使用方法看下面的图片：

1、修改Python脚本中的APIkey

2、修改Python脚本中的文件路径

3、注意Excel文件中sheet页一定要是Sheet1，脚本从这里开始读取数据

4、单元格A1可选MD5、SHA1、SHA256
